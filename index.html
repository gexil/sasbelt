About SASbelt

SASbelt is a Python program developed by G. Exil and K. Wimalan, under the supervision of S. Combet, at the Laboratoire LÃ©on-Brillouin (Saclay, France), building on the MemProt program written by J. Perez and A. Koutsioumpas (Cryst D71:86, 2015, doi:10.1107/S1399004714016678). The program models the low-resolution shape of the amphiphilic belt surrounding a membrane protein, using SAXS experimental data of the belt/protein complex along with a PDB file of the membrane protein structure.

Installation Protocol

To install SASbelt, follow these steps:

  • MS Windows (10, 11)
    • Prerequisites for SASbelt installation
      • Install ATSAS version 2.4.1, ATSAS
      • Install MiKTeX, MiKTeX
      • Install Perl for MS Windows, Strawberry Perl
    • Download and install the SASbelt installer, SASbelt
